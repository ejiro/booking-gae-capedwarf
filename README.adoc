=== CapeDwarf - WildFly / OpenShift

URL on OpenShift : 

* http://capedwarf-mgreau.rhcloud.com/

To build, run

=== App Engine

URL on appengine : 

* https://cloud-booking.appspot.com/

To build, run

    mvn clean package

To start the app, use the [App Engine Maven Plugin]

    mvn appengine:devserver
